# GDB 
Quick usage commands 
gdb --help --> see the usage 

resource: http://www.yolinux.com/TUTORIALS/GDB-Commands.html#GUI 
* build your exe debug
* gdb your.exe --> run your exe in gdb 
* run --> execute your exe 
* bt --> dump the backtrace 
* q --> quit the program
